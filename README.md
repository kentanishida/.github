# Community Health Files

このリポジトリは kentanishida の全リポジトリで共通利用されるコミュニティヘルスファイルを管理します。

## 含まれるファイル

- `.github/ISSUE_TEMPLATE/common_issue.yml` - 統一されたIssueテンプレート

## 使用方法

このリポジトリのファイルは、個別のリポジトリに同名ファイルが存在しない場合に自動的に適用されます。

参考：[GitHub Community Health Files](https://docs.github.com/ja/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file)
